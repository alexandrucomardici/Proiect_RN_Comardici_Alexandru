# Rețeaua Neurală

Aici se află codul pentru:

- `model.py`: definirea arhitecturii multi-task
- `train.py`: antrenarea modelului
- `evaluate.py`: evaluare pe setul de test
- `predict.py`: predicții pe imagini noi
- `utils.py`: funcții auxiliare

Modelul folosește MobileNetV2 ca extractor de caracteristici.
